Task 2: Browser-based javaScript tool

Goal: Demonstrate your ability to design and write web-based applications.

Create a browser-based JavaScript tool for interactive visualizations of mutations in cancer. Your tool will retrieve data from an API provided by the Data Coordination Center of the International Cancer Genomics Consortium.

Task 3: Design novel visualization

Goal: Demonstrate your ability to design a novel visualization.

The task is to propose a visual design for an interactive visualization that allows a user to explore the evolution of a single table over time. Note that you don't need to implement the visualization. We expect a quick literature review in order to find out how others have approached similar research problems before.
